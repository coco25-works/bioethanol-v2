# Market Analysis

Use this skill when the work is comparative and analytical rather than jurisdiction-specific or fully strategic.

## Workflow
1. Define the analytical question.
2. Lock the comparison dimensions.
3. Compare combinations, not just jurisdictions.
4. Isolate legal asymmetries before ranking.
5. State the key finding clearly.
6. Add disqualifier logic, not only upside.
7. Keep the output analytical rather than prescriptive.

## Core Rules
- Explain the comparison logic before showing rankings.
- Use stable rating language.
- Include buyer pull, proof burden, supply fit, and access openness where relevant.
- Include timing, administrative friction, and value-capture mechanics where they create different outcomes across similar markets.
- Add disqualifier conditions for high-priority combinations.
