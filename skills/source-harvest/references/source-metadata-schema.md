# Source Metadata Schema

Capture these fields consistently:
- source title
- source type
- regulator or issuer
- date
- effective date
- in-force status
- amendment or version status
- sunset, grandfathering, or transition window
- jurisdictional level
- language
- official translation status
- citation location
- link
- topic tags
- asymmetry tags
- implementing dependency
- quoted rule text or extraction note
- relevance note
- status

## Status values
- confirmed
- needs translation check
- superseded
- secondary only
- missing
