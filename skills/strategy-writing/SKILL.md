# Strategy Writing

Use this skill only after the evidence and analysis layers exist. This skill turns upstream findings into decisions.

## Workflow
1. Confirm the upstream evidence: market mechanics, product or pathway treatment, and comparative analysis.
2. State the recommendation label early.
3. Explain `Why now`.
4. Name the buyer or counterparty logic.
5. State why the producer, supplier, or operating base can compete.
6. State gating requirements.
7. State better alternatives and stop conditions.
8. Make evidence confidence visible.

## Core Rules
- Write for a commercial / BD reader first.
- Make the buyer explicit.
- Make timing explicit.
- Make the recommendation label explicit.
- Make stop conditions real, not token caveats.
- If a stronger alternative exists, say so directly.

## Output Expectations
- Help the reader decide whether to pursue, defer, monitor, or reject.
- Do not overreach beyond the evidence.
- Do not sound promotional.
