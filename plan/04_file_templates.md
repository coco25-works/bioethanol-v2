# File Template Plan

## Source Register File Template
Use for: `01_source_register/<region>/<jurisdiction>.md`

Required sections:
- Title
- Primary audience
- Decision supported
- Coverage scope
- Primary laws and regulations
- Official guidance and compliance references
- Official data sources
- Source metadata table:
  - source title
  - source type
  - regulator
  - date
  - citation location
  - link
  - topic tags
  - relevance note
  - status
- Open source gaps
- Evidence confidence

- Related files
