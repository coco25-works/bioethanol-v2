# Biofuel Reference

## What This Folder Contains

This folder contains pathway-centered reference files that synthesize regulatory treatment across markets. Each file answers: "For a given pathway, where does regulatory demand exist, how is the pathway treated across major markets, what proof and certification gates apply, and where does Southeast Asian supply fit?"

These files compare across jurisdictions. They do not rank markets or make strategy recommendations.

## How To Read This Folder For Mixed Teams

If you are new to this area, start with [03_biodiesel_and_fame.md](03_biodiesel_and_fame.md) or [04_renewable_diesel_hvo.md](04_renewable_diesel_hvo.md) because they are the clearest examples of how feedstock classification changes commercial value across markets.

The certifications file is important because it shows where proof failures can destroy value.

## How To Use This Folder

- Enter by pathway family: biodiesel, HVO, bioethanol, biomethanol, or biogas/RNG.
- Cross-reference to country files in [../02_country_reference/](../02_country_reference/) for jurisdiction-level detail.
- Use the certifications file as a cross-market gating reference.
- Use feedstock-class files for waste oils, residues, and lignocellulosic context.
