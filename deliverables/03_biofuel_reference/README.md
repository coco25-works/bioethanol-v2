# Biofuel Reference

## What This Folder Contains
This folder contains pathway-centered reference files that synthesize regulatory treatment across markets. Each file answers: "For a given pathway, where does regulatory demand exist, how is the pathway treated across major markets, what proof and certification gates apply, and where does Southeast Asian supply fit?"

These files compare across jurisdictions. They do not rank markets or make strategy recommendations.

### How to read this folder for mixed teams
If you are new to this area, start with `biodiesel_and_fame.md` or `renewable_diesel_hvo.md` because they are the clearest examples of how feedstock classification changes commercial value across markets. The certifications file is important too because it shows where proof failures can destroy value.

## How To Use This Folder
- Enter by pathway family (biodiesel, HVO, bioethanol, biomethanol, biogas/RNG).
- Cross-reference to country files in `../02_country_reference/` for jurisdiction-level detail.
- Use the certifications file as a cross-market gating reference.
- Use feedstock-class files (waste oils, lignocellulosic) for supply-side context.
