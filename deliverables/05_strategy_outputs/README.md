# Strategy Outputs

## What This Folder Contains

This folder contains standalone strategy documents that convert the evidence, reference, and analysis layers into actionable recommendations for a Southeast Asia-based biofuel producer. Each file is a decision memo built on a visible evidence base.

These files recommend. Every recommendation traces back to country references, pathway references, and market analysis.

## How To Read This Folder For Mixed Teams

If you only have time for one file, read [01_executive_strategy_summary.md](01_executive_strategy_summary.md). It gives the portfolio answer in five minutes.

If you need the full route logic, read [02_se_asia_producer_opportunity_map.md](02_se_asia_producer_opportunity_map.md).

If you need to know what not to pursue, read [risk_and_no_go/02_no_go_conditions.md](risk_and_no_go/02_no_go_conditions.md).

## How To Use This Folder

- Start with [01_executive_strategy_summary.md](01_executive_strategy_summary.md) for the portfolio view.
- Use [02_se_asia_producer_opportunity_map.md](02_se_asia_producer_opportunity_map.md) as the center of gravity.
- Enter pathway, market, or producer-type strategy files based on your decision question.
- Use [risk_and_no_go/](risk_and_no_go/) to understand what not to pursue.
