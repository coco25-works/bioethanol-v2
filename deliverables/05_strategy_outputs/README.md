# Strategy Outputs

## What This Folder Contains
This folder contains standalone strategy documents that convert the evidence, reference, and analysis layers into actionable recommendations for a Southeast Asia-based biofuel producer. Each file is a decision memo built on a visible evidence base.

These files recommend. Every recommendation traces back to Phase 2 (country references), Phase 3 (pathway references), and Phase 4 (market analysis).

### How to read this folder for mixed teams
If you only have time for one file, read `executive_strategy_summary.md`. It is designed to give the portfolio answer in five minutes. If you need the full route logic, read `se_asia_producer_opportunity_map.md`. If you need to know what NOT to pursue, read `risk_and_no_go/no_go_conditions.md`.

## How To Use This Folder
- Start with `executive_strategy_summary.md` for the portfolio view
- Use `se_asia_producer_opportunity_map.md` as the center of gravity
- Enter pathway, market, or producer-type strategy files based on your decision question
- Use `risk_and_no_go/` to understand what NOT to pursue
