# Market Analysis

## What This Folder Contains
This folder contains comparative market analysis files that synthesize the country and pathway reference layers into rankings, comparisons, and demand/supply mapping. These files are the analytical bridge between reference (Phases 2-3) and strategy (Phase 5).

These files compare and rank. They do not prescribe go-to-market actions or partnership models.

### How to read this folder for mixed teams
If you only have time for one file, read `global_market_comparison.md`. It gives the overall picture. If you need the route logic, read `feedstock_to_market_mapping.md`. If you need the priority list, read `high_priority_opportunity_ranking.md`.

## How To Use This Folder
- Start with `global_market_comparison.md` for the overall picture
- Use `high_priority_opportunity_ranking.md` for prioritized combinations
- Use `regulatory_asymmetry_matrix.md` for cross-market legal differences
- Use `feedstock_to_market_mapping.md` for SE Asia supply routing
- Use focused market analyses (EU, US, UK, Japan, SE Asia) for deeper dives
