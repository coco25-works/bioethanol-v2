# Market Analysis

## What This Folder Contains

This folder contains comparative market analysis files that synthesize the country and pathway reference layers into rankings, comparisons, and demand/supply mapping. These files are the analytical bridge between reference and strategy.

These files compare and rank. They do not prescribe go-to-market actions or partnership models.

## How To Read This Folder For Mixed Teams

If you only have time for one file, read [01_global_market_comparison.md](01_global_market_comparison.md). It gives the overall picture. If you need the route logic, read [03_feedstock_to_market_mapping.md](03_feedstock_to_market_mapping.md). If you need the priority list, read [02_high_priority_opportunity_ranking.md](02_high_priority_opportunity_ranking.md).

## How To Use This Folder

- Start with [01_global_market_comparison.md](01_global_market_comparison.md) for the overall picture.
- Use [02_high_priority_opportunity_ranking.md](02_high_priority_opportunity_ranking.md) for prioritized combinations.
- Use [03_feedstock_to_market_mapping.md](03_feedstock_to_market_mapping.md) for Southeast Asia supply routing.
- Use [05_regulatory_asymmetry_matrix.md](05_regulatory_asymmetry_matrix.md) for cross-market legal differences.
