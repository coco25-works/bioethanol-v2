# Regulatory Watchlist

**Last updated:** 2026-04-23  
**Purpose:** Track which jurisdictions and regulatory developments need active monitoring.

### Why This File Matters
The strongest strategy in this repository depends on regulatory rules that can change. This file tells the team what to watch so they can react before a recommendation becomes outdated.

## How To Use This Watchlist
- Review quarterly for all Wave 1 jurisdictions
- Review semiannually for Wave 2 jurisdictions
- When a trigger event occurs, initiate the update workflow defined in 08_update_policy.md

## Monitoring Tiers

```text
HIGH VOLATILITY (quarterly + event-driven)
EU / US federal / California / UK / Japan

MEDIUM VOLATILITY (semiannual + event-driven)
Singapore / Brazil

CONTEXT WATCH (periodic or material-change)
```
