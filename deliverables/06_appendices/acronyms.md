# Acronyms

**Last updated:** 2026-04-21  
**Purpose:** Quick-reference list of acronyms used across this repository.

### Why This File Matters
Biofuel regulation is acronym-heavy. This file gives the reader a fast lookup so they do not have to search every time they encounter an unfamiliar abbreviation.
