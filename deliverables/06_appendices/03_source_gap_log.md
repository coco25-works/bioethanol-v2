# Source Gap Log

**Last updated:** 2026-04-23  
**Purpose:** Track unresolved source gaps across all jurisdictions so nothing important is silently missing.

### Why This File Matters
Every claim in the country references and strategy layer depends on source registers. If a source is missing, weak, or translation-dependent, it should be visible here. This file is the team's early warning system for evidence risk.

## How To Use This File
- Every unresolved source issue from jurisdiction source-register files should be logged here.
- Review this file before starting Phase 2 to identify which jurisdictions are blocked.
- Update this file whenever a source gap is resolved or a new gap is identified.
