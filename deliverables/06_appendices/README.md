# Appendices

## What This Folder Contains

This folder contains the operating controls for the deliverables pack: QA status, source gaps, watchlists, versioning rules, glossary material, and writing standards.

Use this section when checking whether the pack is ready to share, what limitations remain, or how future updates should be handled.

## Recommended Use

1. Before sharing the pack, read [00_team_share_readiness_review.md](00_team_share_readiness_review.md) and [01_final_qa_checklist.md](01_final_qa_checklist.md).
2. Before relying on a specific legal or regulatory point, check [03_source_gap_log.md](03_source_gap_log.md), [04_legal_citation_index.md](04_legal_citation_index.md), and [05_regulatory_watchlist.md](05_regulatory_watchlist.md).
3. Before editing or releasing a new version, follow [08_update_policy.md](08_update_policy.md), [09_versioning_rules.md](09_versioning_rules.md), and [12_writing_standard.md](12_writing_standard.md).
