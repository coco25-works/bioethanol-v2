# Country Reference

## What This Folder Contains

This folder contains jurisdiction-level reference files that explain how each market works for bio-based fuels. Each file converts the validated source register into a structured market operating manual.

These files explain market mechanics. They do not rank markets globally or make strategy recommendations.

## How To Read This Folder For Mixed Teams

If you are new to this area, start with Germany or the Netherlands because they are the clearest examples of how compliance value is created for imported biofuels. If you need context on the US market, start with US federal because it shows how a different compliance design works.

## How To Use This Folder

- Enter by jurisdiction under `eu/`, `americas/`, or `apac/`.
- Each file opens with why the market matters and a market-architecture diagram.
- Use the demand mechanism and buyer class sections to understand who pulls fuel into the market.
- Use the asymmetry section to identify which small legal differences should be compared against other markets.
- Cross-reference to [../01_source_register/](../01_source_register/) for evidence detail.

## Wave Structure

Wave 1 demand-center and anchor markets: EU level, Germany, Netherlands, France, US federal, California, UK, Japan.

Wave 2 secondary and contextual markets: Brazil, Singapore, China, Indonesia, Malaysia, Thailand.
