## How Numbering Works

- Root files numbered `00` and `01` are the recommended entry points.
- Folder numbers follow the project build sequence from source evidence to strategy outputs and appendices.
- Files inside each folder are numbered in the recommended reading or operating order.
- `README.md` files are intentionally unnumbered because they are folder landing pages.

## Main Conclusion

**Waste-based HVO/FAME from Southeast Asia to Germany and the Netherlands is the strongest near-term opportunity.**

This conclusion depends on:

1. defensible waste classification,
2. audit-ready documentation,
3. recognized certification,
4. buyer/offtake structures that preserve producer value capture.

## Caveat

This pack is for strategy and decision support. It is not legal advice. Before signing contracts or making investment decisions, verify the latest regulation, guidance, credit pricing, and buyer requirements for the specific route.
